# 🦂 Mirror Scorpion - ميرور سكربيون
> **حيث تُصنع البدايات** | Where Beginnings Are Made

تطبيق متطور مبني باستخدام Flutter يوفر تجربة فريدة تجمع بين أدوات الترجمة المتقدمة والمحتوى الروحي الملهم، مع واجهة مستخدم عصرية ودعم كامل للذكاء الاصطناعي.

---

## 🚀 الميزات الرئيسية

### 🔤 الكارت الأول: ترجمة النصوص (Text Translation)
- **ترجمة فورية**: دعم لأكثر من 100 لغة حول العالم.
- **إدخال صوتي**: تحويل الكلام إلى نص عبر الميكروفون.
- **نطق النصوص**: تحويل النص المترجم إلى كلام مسموع.
- **مشاركة ونسخ**: سهولة مشاركة النصوص المترجمة.

### 💬 الكارت الثاني: الحوار المترجم (Dialogue Translation)
- **محادثة مباشرة**: ترجمة ثنائية الاتجاه للمحادثات الحية.
- **واجهة دردشة**: عرض الحوار بشكل فقاعات دردشة منظمة.
- **دعم كامل للصوت**: إدخال وإخراج صوتي لكلا الطرفين.

### 📄 الكارت الثالث: ترجمة المستندات (Document Translation)
- **التعرف البصري (OCR)**: استخراج النصوص من الصور بدقة عالية.
- **ترجمة الملفات**: معالجة وترجمة المستندات المصورة.
- **تصدير النتائج**: نسخ النصوص المستخرجة أو مشاركتها مباشرة.

### 📖 الكارت الرابع: الأحاديث والقصص والإلهام (Hadith & Stories)
- **المكتبة الروحية**: أحاديث قدسية، قصص الأنبياء، وقصص قرآنية شاملة.
- **إلهام AI**: توليد رسائل تحفيزية مخصصة بناءً على حالة المستخدم النفسية.
- **الاستماع للمحتوى**: دعم النطق الصوتي لجميع القصص والأحاديث.

### 🎮 الكارت الخامس: الألعاب (Games)
- **مكعب روبيك 3D**: محاكاة كاملة للمكعب مع أدوات للحل.
- **شطرنج 3D**: تجربة شطرنج ثلاثية الأبعاد (قيد التطوير).

## 📦 كيفية التشغيل والتطوير

### المتطلبات الأساسية
- Flutter SDK (3.0 أو أحدث)
- Android Studio / VS Code
- Git

### خطوات البدء
1. **استنساخ المستودع**:
   ```bash
   git clone https://github.com/dosoky2580/mirror_scorpion_translate1.git
   ```
2. **تحميل المكتبات**:
   ```bash
   flutter pub get
   ```
3. **تشغيل التطبيق**:
   ```bash
   flutter run
   ```

## 📁 هيكل المشروع (Project Structure)
- `lib/core/`: الثوابت، الثيمات، والودجات المشتركة.
- `lib/features/`: الكروت الستة (الترجمة، الحوار، المستندات، القصص، الألعاب، الإعدادات).
- `lib/services/`: خدمات الذكاء الاصطناعي، قاعدة البيانات، والنطق الصوتي.
- `assets/data/`: ملفات JSON التي تحتوي على الأحاديث والقصص.
